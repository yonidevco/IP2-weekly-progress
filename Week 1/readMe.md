# Students Result PHP Program

## Overview
This program is a simple PHP script that manages and displays a list of students with their scores.  
It calculates whether each student **passes or fails**, finds the **average score**, and identifies the **top-scoring student** and the results are displayed in an **HTML table**.

### It covers the concept of 
        - Functions 
        - Arrays
        - Conditional statemnets
        - Loops
        - PHP functions
        - Variables
